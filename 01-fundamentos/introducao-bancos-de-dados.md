## BANCO DE DADOS

```
-- Uma definição bem simples é uma coleção organizada e estruturada de informações
-- EXEMPLO: Uma agenda de contatos do celular
-- REGISTRO: Cada Contato
-- CAMPO: Nome, telefone, e-email
-- OUTRO EXEMPLO: É a planilha de gastos mensais que no caso é o excel.
```
