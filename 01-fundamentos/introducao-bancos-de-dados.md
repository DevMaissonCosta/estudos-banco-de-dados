## BANCO DE DADOS

```
-- Uma definição bem simples é uma coleção organizada e estruturada de informações
-- EXEMPLO: Uma agenda de contatos do celular
-- REGISTRO: Cada Contato
-- CAMPO: Nome, telefone, e-email
-- OUTRO EXEMPLO: É a planilha de gastos mensais que no caso é o excel.
```

```
 O SGBD -> SIGNIFICA: SISTEMA GERENCIADOR DE BANCO DE DADOS
 BANCO DE DADOS: É ONDE EU TENHO MEUS DADOS ARMAZENADOS, E QUEM PODE MANIPULAR, GERENCIAR OU CONTROLAR ESSES DADOS, É O SGBD

CONCEITOS-CHAVES:
-> DADOS: INFORMAÇOES BRUTAR, NOME NUMEROS OU PALAVRAS.
-> INFORMAÇOES: DADOS ORGANIZADOS E ESTRUTURADOS
-> REGISTRO: LINHA
-> CAMPO: COLUNA
```
### TIPOS BASICOS DE BANCO DE DADOS
```
BANCOS RELACIONAIS (SQL), -> VÃO ARMAZENAR OS DADOS EM TABELAS FIXAS
TABELAS -> (LINHAS/RESGISTROS, COLUNAS/CAMPOS)
EXEMPOS DE SBGDS -> POSTEGRESQL, MYSQL, MICROSOFT SQL SERVER
A LINGUAGEM PARA INTERAGIR COM ESSES BANCOS É O SQL (STRUCTURED QUERY LANGUAGE) -> PERMITI CRIAR, LER E DELETAR FACILMENTE
----
BANCO NÃO-RELACIONAIS (NOSQL) -> VÃO ARMAZENAR EM FORMATO VARIADOS
VARIADOS COMO -> (DOCUMENTOS, CHAVE-VALOR, GRAFOS)
EXEMPLO DE SGBDS PARA NOSQL -> MONGODB, FIREBASE, REDIS
-----
ANALOGIA :
RELACIONAL -> BIBLIOTECA ORGANIZADA
NÃO-RELACIONAL -> CAIXA COM ITENS VARIADOS RELACIONAIS

### INTRODUÇÃO AO SQL
```
SQL (STRUCTURED QUERY LANGUAGE) -> LINGUAGEM ESTRUTURADA QUE PERMITE A CONSULTA DE DADOS EM UM SGBD
UTILIZADO EM BANCOS RELACIONAIS PARA CRIAR, MODIFICAR E COLSULTAR DADOS.
----
PRINCIPAIS COMANDOS SQL:
CREATE TABLE - CRIA TABELAS
INSERT INTO - INSERE DADOS
SELECT - CONSULTA DADOS
UPDATE - ATUALIZA DADOS
DELETE - EXCLUI DADOS
